# CRUD Operations – PHP & MySQL

Basic CRUD-style backend APIs developed for the **Software Engineering Lab** course.  
This work is intended for learning how **PHP interacts with a MySQL database.**

---

## Overview

This practice demonstrates a simple system to manage **Teachers** and **Courses**.

Features included:

- Read Teachers
- Read Courses
- Create Course
- Create Teacher
- Assign Teacher to Course
- Home page with navigation links

---

## Tech Stack

- **PHP**
- **MySQL**
- **XAMPP**
